## 💡 Brand Guidelines

### T-Shirt & Merch Usage
- **Primary:** Deep ocean navy with volt lime accent
- **Works great on:** Dark shirts (navy), light shirts (white/ice)
- **Accent usage:** Volt lime for logos, CTAs, highlights
- **Avoid:** Using volt as primary background (too intense)

### 60-30-10 Rule
- 60%: Deep ocean (backgrounds, headers)
- 30%: White/Ice white (content areas, contrast)
- 10%: Electric volt (buttons, accents, energy)
