# Anton Schneider - Professional Portfolio Website

A modern, professional website showcasing energy modeling work, professional achievements, and technical blog posts.

## Features

✨ **Modern Design**
- Clean, professional interface
- Smooth animations and transitions
- Fully responsive (mobile, tablet, desktop)

⚡ **Interactive Models**
- Live energy demand forecasting tool
- Interactive visualizations
- Real-time parameter adjustments

📝 **Blog System**
- Easy-to-update blog posts
- Clean, readable layout
- Category and date organization

🎨 **Customizable**
- Easy color customization
- Simple content updates
- Modular structure

## Quick Start

1. **Download all files**
2. **Follow DEPLOYMENT_GUIDE.md** for step-by-step instructions
3. **Customize content** to match your information

## Customization

### Update Your Info
Edit `index.html`:
- Lines 49-56: Hero section
- Lines 75-90: About section
- Lines 184-194: Contact info

### Add Blog Posts
Edit `script.js` to add new posts to the `blogPosts` array, then create HTML files in `blog/` folder.

### Change Colors
Edit `styles.css` variables (lines 10-15):
```css
--primary-color: #2563eb;
--primary-dark: #1e40af;
```
